# Geochem Learning Platform

Interactive learning platform for Sedimentary Geochemistry course at FAU Erlangen-Nürnberg.

## Contents

### Learning Modules
- **Module 1**: Introduction to Stable Isotopes
- **Module 2**: Carbon & Oxygen Isotopes
- **Module 3**: Clumped Isotopes
- **Module 4**: Boron Isotopes
- **Module 5**: Strontium Isotopes
- **Module 6**: Nd Isotopes
- **Module 7**: Uranium & Trace Elements

### Exam Simulations
- **Late Devonian Exam**: Frasnian-Famennian mass extinction
- **PTB Exam**: Permo-Triassic boundary mass extinction

## Usage

Open `index.html` in a web browser to access the learning platform.

For GitHub Pages hosting, enable Pages in repository settings and set source to main branch.
